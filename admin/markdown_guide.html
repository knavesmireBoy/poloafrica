• The Markdown Guide by Matt Cone
  • The toolbar above provides some handy shortcuts for creating markdown markup. For a comprehensive guide click the link above.
  • The H button will with each click add a hash to the start of the line, returning to one hash at the seventh succesive click, The result would be a heading from h1 thru h6.
  • The B button will toogle a doulbe pair of asterisks on the selected text. For a single word simply place the cursor anywhwere in the word and click. The result would be to embolden the text.
  • The I button will toogle a single pair of asterisks on the selected text. For a single word see above. The result would be to italicise the text. Already emboldened text will be respected. Therefore my bold text would become my bold text.
  • The ¶ symbol is a hard return and will create a new paragraph within the html. Alternative to the return key.
  • The • adds two spaces to the end of the paragraph creating a line break within the html.
  • The link button creates a link on the selected text. A confirm dialog window opens. Enter a genuine link to either an external website ("https://www.bbc.co.uk/") or an internal web page ("../enquiries/"). To add a title to the link add one space after the hyperlink followed by the title content.
  • The unlink button clears the link. Do select the entire link from bracket to bracket: "[Poloafrica Development Trust][2]" is Uitgedacht Farm.
  • The list button toggles a hyphen on selected lines. The result within the html is an unordered list:
    • • a
    • • b
    • • c
    For an ordered list replace the hypen with '1.'
  • The image button adds an image. All images must have an alt attribute even if it is an empty string. Type in any alterate text, click anywhere in the line, a dialog box will open. Enter the path to the image and an optional title. See link above. Images will be placed at the chosen location. To replace the main image delete the existing image and upload the new one.
  • The arrow button CLEARS all edits.
  • The ? button toggles this guide.
